# Provider Console - Snapshot View

## Overview
Key fields and graphs necessary for a 5 to 10-minute review during provider consultations.

## Data Points and Display Requirements

### Participant Information

| Data Point | Data Source | Recommended Format | Comments |
|------------|-------------|-------------------|-----------|
| Participant Start Date and Program Duration | Clinic data | Start date, Time in program by week # (Table format) | - |
| Participant Lifestyle | Check-in Questionnaire | Table format with response summary | Launched day before appointment/check-in |
| Side Effects | Check-in | If Yes, highlight side effect | Pertinent to visit and archive past. If no side effects, note as "None" |
| Goal Attainment | Stated current goals | Patient's rating of attaining goal | Data found in Check-in |

### Biometric Measurements

| Data Point | Data Source | Recommended Format | Comments |
|------------|-------------|-------------------|-----------|
| Weight | Performed at check-in or Bluetooth scale v2 | Shows initial and current weight, and weight change | Display as "X pounds lost" or "X pounds gained" |
| Waist Circumference | Performed at check-in or virtual v2 | Shows initial and current, and waist circumference change | - |
| Muscle % | Performed at check-in or virtual Bluetooth scale v2 | Shows initial and current, and muscle % change | - |
| Fat % | Performed at check-in or virtual Bluetooth scale v2 | Shows initial and current, and fat % change | - |

### Nutrition Tracking

| Data Point | Data Source | Recommended Format | Comments |
|------------|-------------|-------------------|-----------|
| Food Tracking (kcals) | Upload data | - Days tracked<br>- Average kcals and meal pattern<br>- Pie chart of avg. macronutrients by week<br>- Bar graph of kcals by day/week<br>- Summary of days target met | Based on calculation |
| Macronutrients | Food tracking upload | Carbohydrates, Protein, Fat breakdown | Display as pie chart |

### Lifestyle Factors

| Data Point | Data Source | Recommended Format | Comments |
|------------|-------------|-------------------|-----------|
| Fluid Consumption | Measured by 8 oz cups | - Show avg. daily amount<br>- Days meeting goal<br>- Bar chart and/or time in range graph | - |
| Sleep | Calculated by hours | - Show avg. daily amount<br>- Days meeting goal<br>- Bar chart and/or graph | - |
| Intermittent Fasting | If applicable | - Summary of fasting per week<br>- Hours of fast<br>- Achievement of fast goal<br>- Days meeting target | Show amount of fast per week, length of time |

### Medical Information

| Data Point | Data Source | Recommended Format | Comments |
|------------|-------------|-------------------|-----------|
| Weight Loss Medications | Current dosage and adjustments | - Current dose<br>- Last dose increase by date<br>- Notation of dose adjustment at visit<br>- Graph of dose adjustment over time | - |
| Patient Comorbidities | By ICD10 code date | List format | Notation of dose adjustment at visit |
| Medication List | Active medication list | List format | - |

## Key Requirements for 5-10 Minute Review Workflow

1. **Pre-Visit Preparation**: Check-in questionnaire should be launched the day before the appointment
2. **Visual Priority**: Highlight any reported side effects prominently
3. **Progress Tracking**: Clear visualization of changes from baseline (weight, measurements, etc.)
4. **Goal Achievement**: Easy-to-read summary of patient's self-reported goal attainment
5. **Trend Analysis**: Graphs showing dose adjustments and progress over time
6. **Quick Scanning**: Table and list formats for rapid information retrieval
7. **Actionable Insights**: Clear indicators for when adjustments or interventions may be needed
